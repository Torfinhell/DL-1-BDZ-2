# DL-1-BDZ-2
Implementation of Tranformer model for translation from  french ещ english
To install dependecies:
```
conda create  -n dl-1-bdz-2 python=3.10
```
```
pip install .
```
or
```
pip install uv
uv sync
```
To train model:
```
uv run train.py
```
or
```
python train.py
```
To inference
```
uv run train.py
```
or
